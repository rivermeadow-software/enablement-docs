# Introduction
---
This course introduces you to the RiverMeadow Workload Mobility Platform. You will learn how to migrate workloads from any source to the HVM hypervisor available in HPE Morpheus VM Essentials and HPE Morpheus Enterprise.

### Audience

This course is intended for anyone looking to migrate workloads from on-premises hypervisors or public clouds to HPE Morpheus VM Esssentials.

### Prerequisites

The following prerequisites should be met by learners:

* A basic understanding of HPE Morpheus VM Esssentials

### Learning Objectives

By the end of this course, you will be able to:

* Describe the benefits of the platform's mobility, optimization, and modernization capabilities.
